## How to use? 

### Step 1: Install Docker
- **Windows or Mac:** Install Docker Desktop.  
- **Linux:** Install Docker Engine and the Docker Compose plugin.


### Step 2: Clone the Repository and Enter the Folder

```bash
# Clone the repository
git clone https://github.com/TommyDev-404/grandsight-system

# Go into the project folder
cd grandsight-system
```

### Step 3: Start the Application with Docker:

```bash
docker compose up
```
- Docker will build the images, install dependencies, and set up the database.
- Wait until all services are running before accessing the application.

### Step 4: Access the System:
- Open your browser at,
```bash
http://localhost:5173/login
```

After that, you're good to go. System is now working!
